## Overview

Use this repository to get up and running with the following stack:

Node.js Quickstart

1. Node.js
2. Express.js
3. JavaScript

## Getting Started

Clone this repo and run the following commands from the project root:

1. `npm install`
2. `npm start`
3. Visit http://localhost:3005 to see your app running
